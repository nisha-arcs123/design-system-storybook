Colors

{{ heading }}

Arcon tool is very visual and our color system helps users identify status, see actions, locate help, and to indicate next steps. The consistent use of color in our product keeps cognitive loads low, and makes for a unified and engaging user experience. The colors are designed to be clear and accessible. They come in six color themes.

primary-color

Use this to emphasise main ui components (Tint / #007DB8 / $primary-color)

primary-hover-color

Use only as a hover on primary color (Tint / 20% / 007DB8 / $primary-hover-color )

primary-selected-color

Use this to indicate selected state of primary items (Tint / 70% #007DB8 / $primary-selected-color)

primary-selected-hover-color

Use to indicate hover state on a primary-selected-color items (Tint / 50% / #007DB8 / $primary-selected-hover-color )

primary-highlighted-color

Use to indicate hover state on a primary-selected-color items (Tint 60% #007DB8 / $primary-highlighted-color)

positive-color

Use when you want to indicate sometime positive (success, completion of something...) (Tint / #099138 / $positive-color)

positive-hover-color

Use only as hover color on positive color (Shade / 20% / #099138 / $positive-hover-color)

positive-selected-color

Use only as selected indication for a positive colors (Tint / 60% / #099138 / >$positive-selected-color)

positive-selected-hover-color

Use to indicate hover state on a positive-color-selected items (Tint / 80% / #099138 / 70% $positive-selected-color-hover)

negative-color

Use when you want to indicate a negative action/state (delete, failed action..., error) (Tint 20% / #D72300 / $negative-color)

negative-hover-color

Use only as hover color on negative color (Shade 10% #D72300 / $negative-hover-color)

negative-selected-color

Use only as selected indication for a positive colors (Tint / 80% / #D72300 / $negative-selected-color)

negative-selected-hover-color

Use to indicate hover state on a negative-selected items (Tint / 60% / #D72300 / $negative-selected-color-hover)

warning-color

Use to indicate a warning action/state (severity, alert, caution...) (#FAAD14 / $warning-color)

warning-hover-color

Use only as hover color on warning color (Shade / 10% / #FAAD14 / $warning-hover-color)

warning-selected-color

Use as selected indication for warning colors (Tint / side #FAAD14 / 80% $warning-selected-color)

warning-selected-hover-color

Use to indicate hover state on a warning-selected items (Tint / 60% / #FAAD14 / $warning-selected-color-hover)

private-color

Use when you want to indicate that something is private (board, icons... )

shareable-color

Use when you want to indicate that something is shareable (board, dashboard...) (Tint / 30% / #5F1358 / $shareable-color)

inverted-background-color

Inverted background color (opposite of primary background color) (Shade / 80% / #202020 / $inverted-color-background)

icon-color

Default color for icons (Tint side #53585B $icon-color $icon-color )

icon-active-color

Default color for icons (Tint / 10% / #202020 / $icon-active-color)

icon-color-secondary

Default color for icons (Tint / 30% / #53585B / $icon-secondary-color)

icon-on-dark-bg-color

Default color for icons (Tint / 90% / #202020 / $icon-color-on-dark-bg)

fixed-light-color

Use as color that should remain dark in all themes (Tint / 100 % / #F2F3F5 / $fixed-light-color)

fixed-dark-color

Use as color that should remain dark in all themes ( #202020 / $fixed-dark-color)

Background colors

primary-background-color

Primary background color (Tint / 100 % / #F2F3F5 / $primary-background-color)

secondary-background-color

Secondary background color (Tint / #F2F3F5 / $secondary-background-color)

primary-background-hover-color

Use as hover color (Tints / 90% #53585b / $primary-background-hover-color)

inverted-background-color

Inverted background color (opposite of primary background color) (Tint 80% / #202020/ $inverted-color-background)

grey-background-color

Grey background color (Tint / 20% / #53585b / $grey-background-color)

light-background-color

Grey background color (Tint / 90% /#53585b / $light-background-color)

dark-grey-background

Grey background color (Tint / 40% / #53585b / $dark-grey-background)

dark-background

Grey background color (Shade / 80% / #53585b / $dark-background)

dark-background-hover

Grey background color (Sade 40% / #53585b / $dark-background-hover)

Text colors

primary-text-color

Use for default text color (#202020 / $primary-text-color)

secondary-text-color

Use when you need text with lesser importance (Tint / 40% / #202020 / $secondary-text-color)

secondary-text-on-secondary-color

Use when you need text with lesser importance (on secondary background color) (#F1F3F5 / $secondary-text-on-secondary-color)

text-color-on-inverted

Inverted text color (opposite of primary text color) (#F1F3F5 / $text-color-on-inverted)

text-color-on-primary

Use for text on primary color (#F1F3F5 / $text-color-on-primary)

disabled-text-color

Use as text in disabled components (Tint 70% / #53585B / $disabled-text-color)

placeholder-color

Use for placeholder text in inputs fields (Tint 50% / #53585B / $placeholder-color)

link-color

Use only for links (Tint / 20% / #007DB8 / $link-color)

Border colors

ui-border-color

Border color for ui elements and components (Button, Input...) (Tint / 60% / #53585B / $ui-border-color)

layout-border-color

Border color for general layout and separators (Left pane, Menu Divider...) (Shade / 80% / #53585B / $layout-border-color)

light-border

Border color for general layout and separators (Left pane, Menu Divider...) (Tint / 90% / #53585B / $light-border)

input-hover-color

Border color for general layout and separators (Left pane, Menu Divider...) (#202020 / $input-hover-color)
