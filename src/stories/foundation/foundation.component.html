Colors

{{ heading }}

Arcon tool is very visual and our color system helps users identify status, see actions, locate help, and to indicate next steps. The consistent use of color in our product keeps cognitive loads low, and makes for a unified and engaging user experience. The colors are designed to be clear and accessible. They come in six color themes.

primary-color

Use this to emphasise main ui components (Tint side #017BBA / 0%)

primary-hover-color

Use only as a hover on primary color (Tint side #3397C5 / 20%)

primary-selected-color

Use this to indicate selected state of primary items (Tint side #B3D8E9 / 70%)

primary-selected-hover-color

Use to indicate hover state on a primary-selected-color items (Tint side #80BEDB / 50%)

positive-color

Use when you want to indicate sometime positive (success, completion of something...) (Tint side #219C4B / 10%)

positive-color-hover

Use only as hover color on positive color (Tint side #089137 / 0%)

positive-color-selected

Use only as selected indication for a positive colors (Tint side #84C89B / 50%)

positive-color-selected-hover

Use to indicate hover state on a positive-color-selected items (Tint side #B5DEC3 / 70%)

negative-color

Use when you want to indicate a negative action/state (delete, failed action..., error) (Tint side #D72300 / 20%)

negative-color-hover

Use only as hover color on negative color (Tint side #C22000 / 10%)

negative-color-selected

Use only as selected indication for a positive colors (Tint side #F3BDB3/ 70%)

negative-color-selected-hover

Use to indicate hover state on a negative-selected items (Tint side #EFA799 / 60%)

warning-color

Use to indicate a warning action/state (severity, alert, caution...) (Tint side #FFCB00 / 30%)

warning-color-hover

Use only as hover color on warning color (Tint side #EAAA15 / 0%)

warning-color-selected

Use as selected indication for warning colors (Tint side #FEE6B9 / 70%)

warning-color-selected-hover

Use to indicate hover state on a warning-selected items (Tint side #FDDEA1 / 60%)

private-color

Use when you want to indicate that something is private (board, icons...)

shareable-color

Use when you want to indicate that something is shareable (board, dashboard...) (Tint side #8F5B8B / 30%)

inverted-color-background

Inverted background color (opposite of primary background color) (Tint side #060606 / 80%)

icon-color

Default color for icons (Tint side #363636 / 10%)

fixed-dark-color

Use as color that should remain dark in all themes (Tint side #1A1A1A / 0%)

Background colors

primary-background-color

Primary background color (Tint side #FFFFFF / 100%)

secondary-background-color

Secondary background color (Tint side #F2F3F5 / 0%)

primary-background-hover-color

Use as hover color (Tint side #E6F2F8 / 90%)

inverted-color-background

Inverted background color (opposite of primary background color) (Tint side #060606 / 80%)

grey-background-color

Grey background color (Tint side #EEEEEF / 90%)

Text colors

primary-text-color

Use for default text color (Tint side #202020 / 0%)

secondary-text-color

Use when you need text with lesser importance (Tint side #4D4D4D / 20%)

secondary-text-on-secondary-color

Use when you need text with lesser importance (on secondary background color) (Tint side #FFFFFF / 0%)

text-color-on-inverted

Inverted text color (opposite of primary text color) (Tint side #FFFFFF / 0%)

text-color-on-primary

Use for text on primary color (Tint side #FFFFFF / 0%)

disabled-text-color

Use as text in disabled components (Tint side #989B9D / 40%)

placeholder-color

Use for placeholder text in inputs fields (Tint side #CBCDCE / 70%)

link-color

Use only for links (Tint side #1A8ABE / 20%)

Border colors

ui-border-color

Border color for ui elements and components (Button, Input...) (Tint side #DDDEDE / 80%)

layout-border-color

Border color for general layout and separators (Left pane, Menu Divider...) (Tint side #CBCDCE / 70%)
