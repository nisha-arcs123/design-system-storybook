Radio button in the context of forms refers to a type of input field that allows users to select only one option from a list of predefined choices. Radio buttons are often used when there is a list of mutually exclusive options, meaning the user can select only one option at a time.

A "switch" in the context of forms typically refers to a graphical user interface (GUI) element that represents a binary state, allowing users to toggle between two options - usually "on" and "off". This switch is also known as a toggle switch or a toggle button.
